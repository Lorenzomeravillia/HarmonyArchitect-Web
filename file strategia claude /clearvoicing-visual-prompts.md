## BRAND GUIDELINES (da rispettare in tutti i prompt)

Palette colori:
- Background: #0A0E1A (deep navy/quasi nero)
- Accent primario: #2EC4B6 (teal)
- Accent secondario: #4A90D9 (blue)
- Accent caldo: #F0B429 (amber)
- Accent supporto: #E8873D (orange)
- Accent highlight: #D946A8 (magenta/pink)
- Testo: #F0F2F5 (bianco caldo)

Stile generale:
- Minimal, raffinato, professionale
- Dark theme dominante
- Geometrico e pulito, NO illustrazioni cartoon
- Atmosfera: "strumento professionale per musicisti", non "app giocosa per bambini"
- Ispirazione visiva: Ableton Live, Splice, Native Instruments — tool pro, non giocattolo

## 1. LOGO WORDMARK

### Prompt (per generatore AI o brief per designer):

```
Minimal wordmark logo for "ClearVoicing", a professional music ear training app.

Typography: elegant serif or semi-serif font, modern but with classic music heritage feel. The word "Clear" in white (#F0F2F5), the word "Voicing" in teal (#2EC4B6). 

Style: clean, no icon, no symbol — pure typography. Slight letter-spacing, refined weight. Think Ableton meets classical music program notes.

Background: transparent (for versatile use on dark and light backgrounds).
No gradients on the text. No 3D effects. No drop shadows.

Variations needed:
- Full wordmark: "ClearVoicing" (one line)
- Stacked: "Clear" on top, "Voicing" below
- Compact: "CV" monogram in same style
```

### Dimensioni: SVG vettoriale (scalabile), export anche PNG 2000x600px con sfondo trasparente

## 2. APP ICON (PWA + Favicon + Home Screen)

### Prompt:

```
Minimal app icon for a music ear training application called "ClearVoicing".

Design: abstract representation of sound waves separating into distinct colored streams. 

Concept: a single sound wave on the left side that splits into 4 parallel colored lines/streams flowing to the right, each in a different color: blue (#4A90D9), teal (#2EC4B6), amber (#F0B429), orange (#E8873D). The streams should feel like musical voices separating from a chord.

Background: deep navy (#0A0E1A) with very subtle radial gradient slightly lighter at center.
Shape: square with rounded corners (iOS app icon style, ~22% corner radius).
Style: flat design, no 3D, no skeuomorphism. Clean geometric lines. The streams can have slight glow/bloom effect for depth.

NO musical notes, NO treble clefs, NO piano keys, NO headphones.
The icon should be abstract and modern — recognizable at 32x32px as well as 512x512px.
```

### Dimensioni: 
- 512x512 px (PWA manifest, master)
- 192x192 px (PWA manifest)
- 180x180 px (Apple touch icon)
- 32x32 px (favicon — test leggibilità)
- 16x16 px (favicon piccolo)

## NOTE PER L'ESECUZIONE

### Priorità di produzione:
1. App icon (necessario per PWA manifest — bloccante)
2. Instagram profile picture (necessario per aprire l'account — bloccante)
3. Logo wordmark (necessario per landing page e nav — alta priorità)
4. Open Graph image (necessario per condivisione link — alta priorità)
5. Splash screen visual (sostituisce emoji — media priorità)
6. Reels cover template (necessario per primi video — media priorità)
7. YouTube banner + thumbnail template (quando si apre il canale)
8. Product Hunt graphic (quando si lancia)
9. Social post templates (quando si inizia a postare regolarmente)

### Strumenti consigliati:
- Midjourney o DALL-E per generare i concept base
- Figma o Canva per rifinire, aggiungere testo, creare varianti
- Squoosh.app per ottimizzare i file per web (WebP/AVIF)
- RealFaviconGenerator.net per generare il pacchetto favicon completo dall'app icon

### Formato file:
- Logo: SVG + PNG trasparente
- Icone: PNG trasparente
- Social: PNG o JPG (qualità 90%)
- Favicon: .ico + PNG (generato da RealFaviconGenerator)
